# iPhone-Theme-Maker

A collection of small, independent iPhone customization tools. Each
subfolder is its own self-contained project with its own README.

## Projects

- [`icon-maker/`](icon-maker/README.md) — batch-converts Flaticon PNG/SVG
  icons into themed 1024x1024 iPhone app icons for use as Shortcut icons.
- [`helldivers-info-fetcher/`](helldivers-info-fetcher/README.md) — a local
  test harness plus a working script for a [Widgy](https://apps.apple.com/us/app/widgy-widgets-home-lock-watch/id1524540481)
  text widget that shows live Helldivers 2 Major Order / news info.

See each project's own README for setup and usage.
